    • Overview

    StudyNest is a smart study companion that blends planning, learning, motivation, and performance tracking into a single platform. Designed for students, self-learners, and educators, it integrates advanced features like AI-generated questions, grade analysis, and motivational tools. With an intuitive interface and interactive tools, StudyNest ensures learning stays engaging, personalized, and productive.

    Survey

    Home Page

  • Interactive calendar built with JavaScript
  • Task management using Django database and Task model (date, title, status).
  • Dynamic add/remove tasks via Django forms.
  • Fun CSS celebration animations for completed tasks

    • Video Search Page

    1. YouTube Data API integration to fetch videos based on search keywords.
    2. NLP-based tag matching for better search relevance.
    3. Results displayed in a responsive Bootstrap grid.
    4. Dynamic video playback with JavaScript.

    Flashcard Page

    • PDF upload using Django file handling.
    • Gemini API to extract text and auto-generate questions.
    • Temporary storage of questions in Django models.
    • Interactive flashcards built with JavaScript.
    • Progress tracking via local storage or database.

    Grade Analysis page

    • Django Grade model (subject, date, score).
    • Chart.js integration to visualize grade trends.
    • Dynamic statistics using Django views.
    • Progress bars showing performance improvements.

    Motivation Page

    1. YouTube Data API to fetch motivational videos.
    2. Bootstrap carousel to display content.
    3. Animated text and CSS confetti effects for encouragement.

    Applications

    • Students managing multiple subjects and deadlines.
    • Self-learners tracking learning progress.
    • Educators organizing resources for students.
    • Learners seeking AI-assisted study help.
    • Motivational support for long-term learning goals.

    Future Scope

    • Personalized AI Learning Assistant for custom study paths.
    • Gamification & Rewards to boost engagement.
    • Collaboration & Peer Learning features.
    • AI-generated study notes and summaries.
    • Adaptive quizzes with performance insights.
